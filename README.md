## Описание проекта

Многостраничный сайт интернет-магазина электроники "TechStore" с адаптивной вёрсткой, созданный с использованием HTML5 и CSS3 (без JavaScript).

## Реализованные страницы

• **Главная (index.html)** — приветственный баннер, популярные товары, преимущества магазина  
• **Каталог (catalog.html)** — сетка из 9 карточек товаров с фильтрами  
• **О нас (about.html)** — информация о магазине, миссия и команда

## Реализованные функции

✓ Адаптивное навигационное меню (одинаковое на всех страницах)  
✓ Карточки товаров с hover-эффектами (изображение, название, цена, рейтинг, кнопка)  
✓ CSS Grid для каталога (3 колонки на desktop)  
✓ Flexbox для навигации и футера  
✓ Адаптивная вёрстка (desktop/tablet/mobile)  
✓ Единая цветовая схема и типографика  
✓ Семантическая HTML5-разметка  
✓ Hover-эффекты на карточках, кнопках и ссылках меню

## Технологии

• **HTML5** — семантическая разметка  
• **CSS3** (Flexbox, Grid, Media Queries) — стилизация и адаптивность  
• **Git/GitHub** — система контроля версий
